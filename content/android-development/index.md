Android software development is the process by which applications are created for devices running on Android OS. Google states that "Android apps can be written using Kotlin, Java and C++ languages‚Äù using the Android software development kit (SDK). Since 2017 Kotlin is the official language for Android apps. The official Android app distribution mechanism to end users is Google Play.
